O projeto foi feito em .net 8.0, recomendo o uso de Visual Studio 2022 para executa-lo.
O arquivo de filmes deve ser informado com nome movielist.csv na pasta raiz da solution (mesmo local do arquivo TesteTecnico.sln)
O projeto está usando SQLite como Banco de Dados embarcado.
Antes de executa-lo, certifique-se de que está ajustado para rodar no IIS Express, este item é apenas para evitar problemas com certificados

Qualquer dúvida estou a disposição.
